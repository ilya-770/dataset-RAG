# Функциональные возможности бота

Документ описывает возможности чат-бота «Консультант студента ВИШ»
с точки зрения пользователя. Для информации о том, как управлять
этими функциями через интерфейс администратора, см.
`/wiki/05-admin-panel.md`.

---

## Навигация по разделам

Основной способ взаимодействия с ботом — кнопочное меню. При запуске
(`/start`) пользователь видит главный экран с категориями; каждая
ведёт в подраздел с нужной информацией. При переходе между разделами
предыдущий экран скрывается — в чате не накапливается история
сообщений с устаревшими кнопками.

Текущие категории верхнего уровня:

- Деканат и учебные вопросы
- Стипендии и материальная поддержка
- Проектная деятельность
- АСЭ: трудоустройство и увольнение
- ДМС
- Поступление
- Карты корпусов
- О проекте

> **Открытый вопрос:** по результатам фокус-группы пользователи
> запросили семантический поиск как альтернативу кнопочному меню.
> Решение отложено; подробнее — в ADR-006.

## Шаблоны документов

Бот выдаёт шаблоны часто используемых документов в формате **.docx**
(не PDF — для удобства редактирования без конвертера).

Доступные шаблоны:

- Протокол рабочей встречи
- Анкета для трудоустройства в АСЭ
- Заявление на ПГАС
- Заявление на материальную помощь

Шаблоны хранятся в облаке Росатом; в боте доступны как вложения.
Добавить новый шаблон может администратор контента через админ-панель.

## Регулярные напоминания

Бот автоматически рассылает уведомления подписанным пользователям.

| Напоминание | Дата / периодичность |
|---|---|
| Выплата стипендии | 7-е и 22-е число каждого месяца |
| Сдача отчётности по проекту | Согласно дорожной карте семестра |
| Очные дни в АСЭ | За сутки до планового очного дня |

Отложенная рассылка реализована через брокер очередей. Управление
расписанием — в разделе «Напоминания» административной панели.

## Карты корпусов

Бот отправляет схемы в виде изображений:

- **Карта МИФИ** — расположение корпусов на территории университета;
  по возможности — схема этажей с номерами кабинетов.
- **Карта АСЭ** — план офиса с расположением лифтов, отделов
  и основных точек (HR, бухгалтерия, столовая).

Картинки обновляются аналитиком через облако Росатом;
разработчик вносит новую ссылку в JSON-конфиг раздела.

## Рассылка анонсов мероприятий

Администратор может публиковать анонсы мероприятий ВИШ через
административную панель. Анонс уходит всем подписанным пользователям
немедленно или по расписанию. Пользователи могут подписаться
или отписаться командой `/subscribe`.

---

## Обратная связь

Если пользователь не нашёл нужной информации, бот предлагает
форму обратной связи: сообщение уходит администратору контента.
Функция доступна в любом разделе через кнопку «Написать администратору».

---

## Стилизация контента

Тексты бота поддерживают базовое форматирование Telegram:
**жирный**, _курсив_, __подчёркнутый__. Это улучшает читаемость
и выделяет важные детали (даты, номера кабинетов, суммы выплат).

---

## Что недоступно в текущей версии

- Проверка успеваемости по предметам (требует интеграции с ИС МИФИ).
- Запрос расписания напрямую из бота (требует API образовательной среды).
- Мобильное приложение — только Telegram-клиент (Android и iOS).
  Отдельного нативного приложения нет.
